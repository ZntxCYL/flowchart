# 流程图工具

一个轻量级、纯前端的网页流程图编辑工具，无需安装，打开即用。

## 在线访问

👉 [https://zntxcyl.github.io/flowchart/](https://zntxcyl.github.io/flowchart/)

## 功能

### 概念类型
支持三种概念块：
- **界面** — 方形（矩形），用于表示页面/界面
- **按钮** — 椭圆，用于表示操作/动作
- **条件** — 菱形，用于表示判断/分支

### 节点颜色
提供 5 种预设颜色，点击即切换：
- 🔴 红色 `#e94560`
- 🔵 蓝色 `#3b82f6`
- 🟢 绿色 `#10b981`
- 🟠 橙色 `#f59e0b`
- 🟣 紫色 `#8b5cf6`

### 连线
- 点击工具栏「连线」模式，依次点击两个节点即可创建带箭头的连线
- 选中连线后可点击「删除连线」移除
- 连线颜色固定为灰色 `#6b7280`，与各节点颜色和谐搭配

### 文字编辑
- 双击节点进入编辑模式
- 支持多行文本（回车换行）
- 节点大小根据文字内容自动调整

### 保存与读取
- **保存**：Chrome/Edge 弹出系统文件选择框，可自由选择保存路径和文件名；Safari/Firefox 弹出输入框填写文件名后下载
- **读取**：加载之前保存的 JSON 文件恢复流程图

## 操作方式

| 操作 | 说明 |
|------|------|
| 拖拽节点 | 移动节点位置 |
| 双击节点 | 编辑文字 |
| 点击节点 | 选中节点，可切换颜色/类型 |
| 连线模式 | 依次点击两个节点创建连线 |
| 选中连线 | 可删除连线 |
| Delete 键 | 删除选中的节点或连线 |
| N 键 | 新建节点 |
| C 键 | 切换连线模式 |
| V 键 | 切换选择模式 |
| S 键 | 保存 |
| Esc 键 | 取消当前操作 |

## 技术实现

- 纯 HTML + CSS + JavaScript，单文件应用
- Canvas 2D 渲染
- 深色主题
- 零依赖，无第三方库
